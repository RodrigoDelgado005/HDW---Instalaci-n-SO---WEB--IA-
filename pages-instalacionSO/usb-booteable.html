Creación de USB Booteable

Preparando el medio de instalación

¿Qué es un USB Booteable?

Un USB booteable es una unidad flash USB que contiene un sistema operativo y está configurada para que un ordenador pueda arrancar (boot) desde ella. Se utiliza principalmente para instalar sistemas operativos, como Windows 10.

Requisitos

  • Una unidad flash USB con al menos 8 GB de capacidad
  • La imagen ISO de Windows 10
  • El programa Rufus (para crear el USB booteable)

Pasos para crear un USB Booteable con Rufus

  1. Descarga Rufus desde su página oficial
  2. Inserta tu unidad flash USB en la computadora
  3. Abre Rufus
  4. En "Dispositivo", selecciona tu unidad flash USB
  5. En "Selección de arranque", haz clic en "Seleccionar" y elige la imagen ISO de Windows 10
  6. En "Esquema de partición", elige MBR
  1. Haz clic en "Empezar"
  2. Si aparece un mensaje de advertencia, haz clic en "OK"
  3. Espera a que el proceso termine
Advertencia: Este proceso formateará tu unidad USB. Asegúrate de haber respaldado cualquier dato importante antes de continuar.

Usando la herramienta oficial de Microsoft

Alternativamente, puedes usar la herramienta oficial de Microsoft para crear un medio de instalación:

  1. Descarga la herramienta de creación de medios de Windows 10
  2. Ejecuta la herramienta y acepta los términos de licencia
  3. Selecciona "Crear medios de instalación para otro PC"
  1. Elige la versión de Windows y la arquitectura (32 o 64 bits)
  2. Selecciona "Unidad flash USB" como medio
  3. Selecciona tu unidad USB
  4. Espera a que se complete la descarga y la creación del medio de instalación
Consejo: La herramienta de Microsoft es más fácil de usar, pero Rufus ofrece más opciones de configuración avanzada.
